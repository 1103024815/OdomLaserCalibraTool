# laser-odom_calibration

## 1. Prerequisites

1. ubuntu 16.04 + ROS Kinetic

2. Eigen3

## 2. Usage

```
    cd catkin_ws/src
    git clone https://github.com/MegviiRobot/OdomLaserCalibraTool
    cd ..
    catkin_make
    source devel/setup.bash
    roslaunch example.launch
```
